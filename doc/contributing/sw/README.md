# Contributing to Software

Software is an important part of testing and deploying Pavona devices.
One important distinction when discussing software is that there are multiple [cores](../../../hw/doc/cores.md) in Pavona that can run software, as well as software that runs on a connected host computer for test and development purposes.
For more information about contributing to Pavona software see the following pages:
- [Device Interface Functions](./device_interface_functions.md)
- [Writing and Building Software for ACC](./otbn_sw.md)
